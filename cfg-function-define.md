# Objeto: **define**
- define un parámetro de la función


## Parámetros
#### type
- en este caso debe ser `type="param"`.

#### id
- identificador del parámetro.
- de preferencia deben ser minúsculas en formato `kebab-case`.

#### defaultValue
- valor por omisión del parámetro
- puede ser una expresión.

#### notNull (true, false)
- en caso de valores nulos se genera un 0 (cero) o `''` (texto vacío), dependiendo del tipo de datos.
