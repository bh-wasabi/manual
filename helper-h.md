# Markup: **h1..h6**
- pone un bloque como encabezado (header)
- h1, h2, h3, h4, h5, h6

## Contexto
- no tiene

## Parámetros
#### class
- clases CSS a utilizar
- se pueden agregar multiples clases separadas por espacio.

#### align
- `left`
- `right`
- `center`
- `justify`

#### noWrap (true, false)
- elimina el ajuste de línea (word wrap=) del bloque de texto.
