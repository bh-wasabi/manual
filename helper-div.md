# Markup: **div**
- genera un bloque generico

## Contexto
- puede llevar un contexto opcional

## Parámetros
#### class
- clases CSS a utilizar
- se pueden agregar multiples clases separadas por espacio.

#### align
- `left`
- `right`
- `center`
- `justify`

#### noWrap (true, false)
- elimina el ajuste de línea (word wrap=) del bloque de texto.
