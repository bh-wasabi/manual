# Markup: **panel**
- pone un panel en la página

## Contexto
- no tiene

## Parámetros

#### name
- nombre visible a mostrar

#### class
- clases CSS a utilizar
- se pueden agregar multiples clases separadas por espacio.

#### align
- `left`
- `right`
- `center`
- `justify`

#### noWrap (true, false)
- elimina el ajuste de línea (word wrap=) del bloque de texto.
