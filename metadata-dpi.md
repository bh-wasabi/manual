#### document
- tipo de documento

#### requestType
- tipo de solicitud

#### requestSubType
- sub tipo solicitud

#### subType
- sub tipo de documento

#### service
- servicio

#### attribute1
- atributo 1 del documento

#### attribute2
- atributo 2 del documento

#### attribute3
- atributo 3 del documento

#### path
- ruta base para las expresiones

#### subPath
- sub ruta base para las expresiones

#### condition1
- se pueden establecer condiciones para su ejecución

#### condition2
- se pueden establecer condiciones para su ejecución

#### condition3
- se pueden establecer condiciones para su ejecución

#### lastChange
- columna informativa para registrar el último cambio registrado manualmente

#### ignore (booleano)
- es posible comentar los renglones del metadata usando esta columna, tambien se recomienda poner el color de las letras en gris para que sea más visual lo comentado.

#### include
- lista de proyectos (separados por comas) que se desean incluir.

#### exclude
- lista de proyectos (separados por comas) que se desean excluir.

#### type
- tipo de DPI

#### set
- sección donde corresponde el DPI

#### group
- grupo al que corresponde el DPI

#### key
- campo llave

#### keyName
- nombre del campo llave
- puede ser una expresión

#### unit
- nombre de la unidad opcional

#### codeInternational
- codigo internacional

#### valueText
- valor texto (si aplica)

#### valueNumber
- valor numérico (si aplica)

#### valueNumber2
- algunos DPIs pueden tener 2 valores numéricos

#### valueBoolean
- valor lógico si aplica

#### valueName
- nombre del valor

#### service
- servicio que genero el DPI

#### serviceName
- nombre del servicio que genero el DPI

#### formatTime
- formato de hora opcional

#### dateTime
- se puede establecer la fecha hora con otra columna

#### time
- se puede indicar una hora especifica

#### proyectTimes
- es posible generar una explosión del DPI usando esta columna

#### proyectUnit
- unidad de explosión

#### proyectSpeed
- velocidad de explosión

#### request
- solicitud

#### requestName
- nombre de la solicitud
