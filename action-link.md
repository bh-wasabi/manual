# Objeto: **link** (action)
- ejecutar un vínculo desde una acción

## Parámetros

#### href
- vínculo interno o externo a abrir.
- puede ser una [expresión](expr.md).

#### name
- nombre a mostrar al abrir un vínculo.
- puede ser una [expresión](expr.md).

#### title
- título del tab a mostrar al abrir un vínculo.
- puede ser una [expresión](expr.md).

## Sub objetos
#### [param](param.md)
- es posible pasar parámetros adicionales.
- estos parámetros se generan como parte del vínculo nuevo.
