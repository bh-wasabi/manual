# Objeto: **config**
- define un `preset` fijo del sistema
- es posible agregar otros campos como en los otros preset's.

## Parámetros
#### id
- identificador del `preset`

#### nombre
- nombre del `preset`

#### source
- es posible generar este `preset` a partir de una vista especifica.

#### view
- es posible generar este `preset` a partir de una vista especifica.

## Sub objetos
#### item
- debe contener por lo menos `id` y `nombre`

#### default
- valores por omisión para no repetir en cada `item`.
