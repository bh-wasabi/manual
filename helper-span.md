# Markup: **span**
- sirve para cambiar algunas propiedades de un bloque de texto.

## Contexto
- no tiene

## Parámetros
#### class
- clases CSS a utilizar
- se pueden agregar multiples clases separadas por espacio.

#### align
- `left`
- `right`
- `center`
- `justify`

#### noWrap (true, false)
- elimina el ajuste de línea (word wrap=) del bloque de texto.
