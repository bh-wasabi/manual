# Objeto: **ask**
- pregunta un parámetro manualmente.

## Parámetros

#### field
- nombre del campo

#### type
- `date` pregunta una fecha
- `text` por omisión un texto.

#### mode
- email 
- password
- search
- tel
- text
- url

#### label
- nombre a desplegar
- puede ser una [expresión](expr.md).

#### value
- valor por omisión
- puede ser una [expresión](expr.md).

#### required (true, false)
- define si el parámetro es requerido
