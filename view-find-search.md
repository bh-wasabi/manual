# Objeto: **search**
- define los campos por los que se va efectuar la búsqueda.

## Parámetros
#### condition 
- es posible definir una condición previa

#### field
- ruta del campo dentro del documento, normalmente es "seccion.campo"

#### directSearchIfLength (#)
- si el valor es de la longitud explicita busca únicamente por este campo

#### type
- text (por Omisión)
- number, busca el valor numérico
