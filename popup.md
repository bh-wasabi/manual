# Objeto: **popup**
- abre una ventana tipo popup.

## Parámetros
##### title
- titulo, puede ser una [expresión](expr.md) sobre el contexto actual.

##### fullScreen (true, false)
- usa toda la pantalla

##### source
- opcional podemos indicar un tipo de documento especifico.
- puede ser una [expresión](expr.md) sobre el contexto actual.

##### sourceId
- opcional podemos indicar un id especifico
- puede ser una [expresión](expr.md) sobre el contexto actual.

##### sourceView
- opcional podemos una vista especifica
- puede ser una [expresión](expr.md) sobre el contexto actual.

## Sub objetos
##### [param](param.md)
- para pasar parámetros a la vista.
