# Objeto: **field** (table)
- define un campo de la tabla SQL.

## Parámetros
#### id
- identificador del campo

#### type
- `text` campos tipo texto (por omisión)
- `number` campos numéricos
- `date` fechas
- `boolean` lógicos (true, false)

#### notNull (false, true)
- define si el campo acepta o no valores nulos.

#### case
- `upper`
- `lower`
- `capitalize`
