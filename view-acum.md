# Objeto: **acum** (view)
- definir un campo calculado acumulado sobre el resultado de una consulta.

## Parámetros

#### field
- nombre del campo a definir.

#### value
- [expresión](expr.md) a calcular usando el contexto del resultado unitario de la consulta.

#### type
- `sum` 

#### direction
- `asc` ascendente (por omisión).
- `desc` descendente.
